# 🧩 Extensão Secundária - Transformador de Texto 

A mesma ferramenta robusta de manipulação de strings, agora totalmente repaginada com uma nova identidade visual, paleta de cores moderna, tipografia refinada e layout focado em UX.

---

## 🎯 Objetivo do Projeto

A **Extensão Secundária** mantém seu propósito principal intacto: oferecer um utilitário ultrarrápido para formatar e transformar textos (converter caixas, aplicar filtros e tratamentos de string) diretamente pelo popup do navegador. 

O foco desta versão foi uma **reestruturação completa de design**:
- **Nova Paleta de Cores:** Contraste aprimorado e visual moderno.
- **Layout Otimizado:** Componentes reposicionados para maior conforto visual e usabilidade (UI/UX).
- **Tipografia Atualizada:** Escolha de fontes que melhoram a leitura do texto transformado.
- **Novos Assets Visuais:** Renovação completa do logotipo e favicon da extensão.

---
## 🛠️ Tecnologias 
-   **Manifest V3**  (O padrão mais moderno, seguro e performático de extensões)
-   **Web nativa:**  HTML5, CSS3 e JavaScript

---

## ⚙️ Instalação Rápida

1. Baixe e extraia o arquivo `Extensao_Secundaria-main.zip`.
2. Abra o seu navegador baseado em Chromium (Chrome, Brave, Edge, Opera).
3. Acesse a página de extensões através do endereço `chrome://extensions/`.
4. No canto superior direito, ative o **"Modo do desenvolvedor"**.
5. Clique em **"Carregar sem compactação"** (Load Unpacked) no canto superior esquerdo.
6. Selecione a pasta extraída `Extensao_Secundaria-main` (que contém o arquivo `manifest.json`).
